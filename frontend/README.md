# Drones España - Zonas de Vuelo

App web para pilotos de drones en España. Busca una dirección y revisa en el mapa las zonas donde puedes volar, con códigos de colores según las restricciones oficiales de ENAIRE.

## Instalación

### Frontend
```
cd frontend
npm install
npm start
```

### Backend
```
cd backend
npm install
npm start
```

## Uso
- Busca una dirección.
- El mapa mostrará un radio de 1km y las zonas UAS según ENAIRE.
- Verde: sin restricciones. Amarillo: aviso o restricción parcial. Rojo: prohibido.
